Install Ubuntu on the Orange Pi Zero

Install Ubuntu Core

We will walk you through the steps of flashing Ubuntu Core on an Orange Pi Zero. At the end of this process, you will have a board ready for production or testing snaps.

Minimum requirements

  • An Ubuntu SSO account with an SSH key
  • An Orange Pi Zero
  • A microSD card
  • A USB to TTL Serial Cable
  • An Ethernet cable (for first setup)
  • A 5V power supply rated at 2 amps and a cable with a micro USB plug
  • screen installed on your host computer (sudo apt install screen)

  1. 2 Download Ubuntu Core

    Get the correct Ubuntu Core image for your board:

    • Ubuntu Core 16 for Orange Pi Zero
    2. {% include "./_flash-image.html" with card="microSD card" number=3 %}

  2. 4 Install Ubuntu Core

    1. Insert the SD card into the board.
    2. Connect the Ethernet cable to the board and to the network.
    3. Connect the TTL end of the TTL to USB cable, to the three pins next to the Ethernet port. On this diagram, they are identified by the label "Debug serial port". The pin layout is, from the outside to the inside:
      • GND (black wire)
      • RX (white wire)
      • TX (green wire)
    4. Connect the USB end of the cable into your host computer, open a terminal and run sudo screen /dev/ttyUSB0 115200 to open a listening connection to the board.
    5. Power on the board. The terminal on your host computer should display the boot sequence. If not, please ensure the TTL to USB cable is correctly connected.
