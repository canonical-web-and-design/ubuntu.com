Install Ubuntu Core on the Samsung Artik 5 or 10

There are two install options for the Samsung Artik: Ubuntu Core or Ubuntu Server. This page is for Ubuntu Core.

Install Ubuntu Core

We will walk you through the steps of flashing Ubuntu Core on a Samsung Artik 5 or 10. At the end of this process, you will have a board ready for production or testing snaps.

Minimum requirements

  • An Ubuntu SSO account with an SSH key
  • A Samsung Artik 5 or 10
  • An SD card
  • An Ubuntu Core image

Installation instructions

    {% include "./_setup-ubuntu-sso.html" with number=1 %}

  1. Download Ubuntu Core

    Get the correct Ubuntu Core image for your board:

    • Ubuntu Core 16 image for Samsung Artik 5
    • Ubuntu Core 16 image for Samsung Artik 10
    • You can verify the integrity of the file using the MD5sum file.
    2. {% include "./_flash-image.html" with card="SD card" number=3 %}

  2. Install Ubuntu Core

    1. Prepare your Artik to boot from the SD card, by setting the "SW2" switches to 1:on and 2:on
    2. Insert the SD card
    3. Connect the 5V power supply to the board, then use the power button on the board to switch it on
    3. {% include "./_first-boot-setup.html" with number=5 %} {% include "./_login.html" with number=6 %}
