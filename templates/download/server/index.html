Ubuntu for ARM ›

Ubuntu Server and Ubuntu Desktop for ARM are a general purpose OS for ARM-based systems. A variety of SoC platforms are certified. Please note that currently only Calxeda ECX-1000 (Server only), Marvell Armada-XP (Server only) and TI Panda development boards are supported.
