Installing Ubuntu OpenStack

The best way to deploy Ubuntu OpenStack is with MAAS and Juju. Here is the recommended way to do it.

Ubuntu OpenStack is one of the many workloads you can deploy using Ubuntu Server. A full deployment will require a minimum of 10 servers. Please note, you will need to install Juju twice, first on top of MAAS to deploy Ubuntu OpenStack and then again on Ubuntu OpenStack to deploy your workloads. Here is a summary of the steps you will need to follow:

  1. 1

    Download Ubuntu Server for Cloud ›

  2. 2

    Burn a DVD or create a bootable USB stick.

  3. 3

    Install it and select ‘MAAS’ when asked which workload to deploy.

    More information ›

  4. 4

    Add six or more nodes to MAAS.

    More information ›

  5. 5

    Deploy Juju on MAAS.

    More information ›

  6. 6

    Get the latest Ubuntu OpenStack release from the Ubuntu Cloud Archive.

    sudo add-apt-repository cloud-archive:icehouse

    More information ›

  7. 7

    Deploy Ubuntu OpenStack using Juju.

    More information ›

  8. 8

    Configure Juju for OpenStack.

    More information ›

  9. 9

    Start deploying services to your cloud with Juju.
