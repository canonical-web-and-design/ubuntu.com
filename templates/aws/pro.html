What's in Ubuntu Pro?

  • FIPS & CC-EAL2 certified

    Certified components to enable operating environments under compliance regimes like FedRAMP, HIPAA, PCI and ISO.

  • CIS and DISA STIG profiles

    For companies looking to leverage industry benchmarks for hardening, Ubuntu Pro makes two leading profiles available out of the box.

  • Open source security

    Ubuntu Pro adds security coverage for the most important open source applications like Apache Kafka, NGINX, MongoDB, Redis and PostgreSQL.

  • Predictable and Secure

    Ubuntu Pro is based on Ubuntu and leverages both the breadth of open source included and the AWS optimisations built in.

  • Kernel Livepatch

    Ubuntu kernel updates are regularly issued. With our Livepatch service, kernel patches are delivered immediately, without the need for reboot.

  • Fully AWS-integrated

    Starting in H2 2020, Ubuntu Pro will include integrations with AWS services including Security Hub, CloudWatch and CloudTrail.

  • The right Ubuntu for you

    Canonical brings the three most popular Ubuntu Server distributions to choose from - 14.04 LTS, 16.04 LTS, 18.04 LTS and 20.04 LTS.

  • 10-year lifetime

    Canonical backs Ubuntu Pro for 10 years*, ensuring security updates are available throughout, with a guaranteed upgrade path.

  • Priced for the cloud

    Ubuntu Pro pricing tracks the underlying EC2 cost, starting at 20% for the smallest t3.nano instances, and ramping down to less than 1%.

* For 18.04 LTS onwards. The maintenance period for Ubuntu Pro 14.04 LTS and 16.04 LTS is 8 years.
